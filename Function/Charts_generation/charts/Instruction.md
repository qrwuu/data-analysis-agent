# 数探 Agent · 使用说明

> 面向商业分析师的智能数据分析平台 — 上传 Excel / 连接数据库 / 调用 Google Sheets / 自定义 API，然后用自然语言提问，AI 自动写 SQL、跑分析、画图、生成报告。

---

## 🚀 快速开始

1. **配置一个模型**：右上角侧栏 → 模型行右侧的 ⚙ → 填入 API Key（DeepSeek / OpenAI / Claude 任选一个，或添加自定义 OpenAI 兼容模型）
2. **添加数据源**：侧栏「＋ 添加数据源」→ 选择 Excel / SQL / Google Sheets / 自定义 API
3. **开始提问**：在底部输入框直接用自然语言问，例如：
   - `各品类的销售额是多少？`
   - `近 12 个月的销售趋势如何？`
   - `哪些客户贡献了最多收入？`

## ⚡ 斜杠命令

输入 `/` 调起命令面板。按分组：

### 分析 · 图表
| 命令 | 用途 |
|------|------|
| `/chart`      | 强制生成图表（从 41 种里选最匹配的） |
| `/sql`        | 直接执行 SQL，结果格式化为表格 |
| `/decile`     | 十分位分桶 + Pareto 曲线 |
| `/tree`       | 决策树（ID3 / C4.5 / CART）+ 特征重要性 + ROC |
| `/kmeans`     | K-Means++ 聚类 + 肘部曲线 |
| `/logistic`   | 逻辑回归（OvR 多分类）+ 系数重要性 + 混淆矩阵 |
| `/regression` | 线性 / 岭回归 + t 检验 + 残差诊断 |

### 时间序列
| 命令 | 用途 |
|------|------|
| `/arima`   | ARIMA + AIC 自动选阶 + 置信区间 |
| `/sarima`  | 季节性 ARIMA + 自动周期检测 |
| `/var`     | 多变量向量自回归 + 格兰杰因果 |
| `/prophet` | Prophet 加法分解（趋势 + 季节性 + 变点） |
| `/gru`     | GRU 深度学习预测 + MC Dropout 95% 置信区间 |

### 清洗
| 命令 | 用途 |
|------|------|
| `/data`      | 数据概况：缺失率 / 分布 / 极值 |
| `/inset`     | 缺失值处理（补 0 / 均值 / 中位数） |
| `/winsorize` | 分位数缩尾（截断极端值） |
| `/trimming`  | 截尾（保留 min~max 范围内的行） |

### 导出
| 命令 | 用途 |
|------|------|
| `/export`    | 导出为 Excel（消息中需含「导出」才执行） |
| `/report`    | 生成 Word 分析报告（消息中需含「导出」才执行） |
| `/ppt`       | 生成麦肯锡风格 PowerPoint |
| `/dashboard` | 生成可交互销售分析看板 |

### 工具
| 命令 | 用途 |
|------|------|
| `/status` | 查看当前模型 / 数据源 / Token 用量（不发起 API 请求） |

## 🗂 数据源

| 类型 | 说明 | 关键参数 |
|------|------|---------|
| **Excel / CSV** | 本地文件上传，calamine 引擎解析 | `.xlsx` `.xls` `.csv` |
| **SQL 数据库** | SQLAlchemy 连接字符串 | 例如 `mysql+pymysql://user:pass@host:3306/db` |
| **Google Sheets** | 服务账号 JSON + 表格 URL/ID | 需将服务账号邮件加为表格协作者 |
| **HTTP API** | 任意 REST 端点 | URL + 可选认证（Bearer / X-API-Key） |

## 💡 使用技巧

### 多步分析
直接说自然语言即可，Agent 会自动：
1. `get_schema` 查看表结构
2. `query_data` 写 SQL 探数
3. `create_analysis_table` 准备好聚合表（多表 JOIN / 字段重组时）
4. `select_chart` → `generate_chart` 出图

### 两阶段确认
导出类（PPT / Excel / 报告 / 看板）会先生成**大纲**让你确认，再实际生成。可以：
- 点 **✅ 确认生成** 直接出文件
- 点 **✏️ 修改大纲** 用自然语言指示修改（如"把第 3 张换成双栏文字"）
- 点 **✕ 取消** 放弃

### 数据预览
右上角「数据预览」按钮可弹出表格视图，按 sheet 切换，最多看 100 行。

### 已保存对话
侧栏底部「💾 保存」可命名当前对话，下次想接着分析就在「已保存对话」列表点开。数据源会自动重连（若文件还在）。

### 业务知识库
侧栏「🧠 业务知识库」管理：
- **指标定义**：DAU 怎么算、SQL 模板
- **业务规则**：合理性约束（如「次日留存不能超过首日 DAU」）
- **背景知识**：分析常识（如「分析流失时需检查版本更新」）

Agent 会自动参考这些定义。

### MCP 工具
侧栏「MCP」可挂接 Model Context Protocol 服务器（如 Playwright 浏览器、文件系统等），让 Agent 调用外部工具。

## 🛠 故障排查

| 现象 | 原因 / 解决 |
|------|------------|
| LLM 报 401 | API Key 错误；侧栏 ⚙ 重新填入 |
| LLM 报 429（限流） | 自动重试 3 次（5s/15s/30s）后失败；稍后再试或换 provider |
| 图表显示不出 | 通常是 LLM 写的 SQL 错；查看工具步骤的具体错误 |
| 数据源「已断开」但功能仍可用 | 上次会话残留；点「刷新」或新建会话即可 |
| 长对话回复变慢 | 接近 context 上限时自动触发压缩（紫色 ✦ 步骤）；正常现象 |
