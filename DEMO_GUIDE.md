# 数探 Agent · 5 分钟体验指南

这条路径使用仓库内的匿名示例数据，不需要准备真实业务文件。

## 1. 启动

```powershell
# Windows
.\.venv\Scripts\Activate.ps1
python app.py
```

```bash
# macOS / Linux
source .venv/bin/activate
python app.py
```

打开 <http://localhost:5001/>。

## 2. 进入工作台

交付版本已由服务端接入 AI 能力。终端用户无需填写接口地址、模型名称或访问密钥，打开工作台即可继续。自行部署时，模型服务由管理员在服务端统一接入。

## 3. 加载示例数据

点击输入框左侧「添加数据」→「使用示例数据」。加载完成后，顶部会显示当前数据源。

打开「数据预览」，浏览数据表、字段和样例行，并确认本轮分析需要使用的表。

## 4. 完成三次分析

依次发送：

```text
先检查数据质量，列出缺失值、重复记录和明显异常值。
```

```text
按月汇总销售额，生成趋势图，并指出变化最大的月份。
```

```text
比较各商品的销售贡献和退款表现，给出三条有数据依据的行动建议。
```

在回答中可以看到流式文本、工具执行状态、表格和图表。点击建议问题可继续追问，也可以使用「停止」中断当前生成。

## 5. 体验显式工具

点击「分析工具」，搜索并选择以下任一能力：

- `chart`：根据描述生成图表；
- `sql`：执行只读查询；
- `kmeans`：完成客户或商品分群；
- `report`：生成分析报告；
- `dashboard`：生成可交互仪表盘。

也可以在输入框键入 `/`，使用 `/data`、`/skills`、`/jobs`、`/knowledge` 和 `/sessions` 等快捷命令。

## 6. 保存结果

- 从图表卡片复制或下载图片；
- 从任务卡片下载 Excel、报告、PPT 或其他产物；
- 使用 `/jobs` 查看后台任务；
- 登录后保存当前分析，并在「历史分析」中恢复。

完成这条流程后，已经覆盖数据接入、上下文确认、AI 分析、工具调用、结果验证和产物交付的完整闭环。
