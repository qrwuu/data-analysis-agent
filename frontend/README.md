# Frontend source layout

All maintained chat-application JavaScript lives under `frontend/`. The files in `static/dist/` are production artifacts generated by Vite and should not be edited by hand.

## Structure

- `core/` — DOM helpers, application state, themes, overlays, API access, and compatibility bridges.
- `entries/` — Vite entry points for the chat application and dashboard.
- `features/` — user-facing modules such as models, skills, workspaces, teams, and lazy-loaded Vue UI islands.
- `legacy/` — maintained compatibility modules that are being loaded through the modern ESM entry point.

## Development

```bash
pnpm install
pnpm format:check
pnpm lint
pnpm build:check
```

The Flask template loads `/static/dist/chat-app.js`; run the build after changing frontend source so the checked-in production bundle stays in sync.
